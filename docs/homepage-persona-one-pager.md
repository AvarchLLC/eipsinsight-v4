# Homepage Plan by Persona 

This is a simple one-page view of what each homepage should show.

Date: April 1, 2026

Included personas:
- Developer
- Editor
- Builder
- Newcomer

---

## Developer Homepage

### Goal
Help developers quickly see what is changing and what needs action.

### What it will show
- Upgrade Watch
- Trending Proposals
- Browse by Status, Category, and Repository
- Governance Flow Highlights
- Tool Shortcuts (Board, Timeline, Dependencies)
- Recent Activity

### What it will remove
- Heavy learning content at the top
- Extra promotional sections

---

## Editor Homepage

### Goal
Help editors focus on review workload and proposal movement.

### What it will show
- Editor Review Queue
- Browse by Status, Category, and Repository
- Governance Flow Highlights
- Recent Review and Status Activity
- Tool Shortcuts

### What it will remove
- Beginner learning blocks
- Non-essential marketing content

---

## Builder Homepage

### Goal
Help builders discover active standards and contribute faster.

### What it will show
- Trending Proposals
- Browse by Status, Category, and Repository (ERC-focused)
- Tool Shortcuts (EIP Builder first)
- Recent Activity
- Practical Resources

### What it will remove
- Editor-only queues
- Dense analytics blocks at the top

---

## Newcomer Homepage

### Goal
Make Ethereum standards easy to understand and easy to start.

### What it will show
- Learning Resources (first)
- Trending Proposals (simple cards)
- Upgrade Watch (simple summary)
- Beginner-friendly Tool Shortcuts

### What it will remove
- Complex charts in first view
- Deep technical tables
